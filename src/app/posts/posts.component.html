Post 1 - Packages In Javascript

So what exactly are Packages?
Packages are a set of functions and logics written by someone else for us to use.
To install packages we use the `npm install [package name]` or `npm i [package name]` command.

For Example: 

 npm install colors

Post 2 - Objects in Javascript

You may be wondering, what exactly are objects.
Objects are like suitcases that can contain other objects or key-value pairs.

For example: 

 var person1 = { 
 name: 'John', 
 age: 26, 
 marks: { 
  english: 22, 
  math: 23, 
  science: 24 
 }
}
